# java-Programming

Repositório destinado ao estudo e prática dos fundamentos da linguagem Java, por meio de exercícios e desafios resolvidos durante o curso da Udemy **"Mastering Java Basics"**.

## Curso de referência

Os códigos presentes neste repositório foram adiquiridos e desenvolvidos ao longo do curso, com alguns deles sendo "notes" disponibilizadas pelo instrutor:

- [Mastering Java Basics (Udemy)](https://www.udemy.com/course/mastering-java-basics/)

O objetivo principal é reforçar o aprendizado dos conceitos básicos de Java por meio de implementação prática.

## Objetivo do repositório

- Consolidar conhecimentos fundamentais de Java.
- Praticar a sintaxe e os principais recursos da linguagem.
- Organizar exercícios e desafios resolvidos ao longo do curso.
- Servir como material de revisão e consulta futura.

## Conteúdos abordados

Alguns dos principais tópicos trabalhados neste repositório incluem:

- Fundamentos da linguagem Java
  - Sintaxe básica
  - Tipos primitivos
  - Variáveis e constantes
  - Operadores aritméticos, relacionais e lógicos
- Estruturas de controle
  - `if`, `else if`, `else`
  - `switch`
  - Laços `for`, `while`, `do-while`
- Arrays e matrizes
  - Declaração, inicialização e iteração
  - Arrays unidimensionais
  - Matrizes (arrays multidimensionais)
- Strings
  - Criação e manipulação de strings
  - Métodos comuns da classe `String`
  - `StringBuilder` e `StringBuffer` (quando aplicável)
- Programação orientada a objetos (POO) (SUPERFICIAL)
  - Classes e objetos
  - Atributos e métodos
- Coleções (Collections)
  - `List`, `Set`, `Map`
  - Iteração com `for-each`
